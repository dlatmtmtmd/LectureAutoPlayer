# Lecture Auto Player

TLS에서 강좌의 모든 강의 동영상을 자동으로 재생&스킵하여 완료도 100%로 만들어주는 프로그램입니다.

## 📥 다운로드 (가장 쉬운 방법)

**[Releases](https://github.com/dlatmtmtmd/LectureAutoPlayer/releases)** 페이지에서 `LectureAutoPlayer.exe` 파일을 다운로드하여 바로 실행하세요.

- Python 설치 불필요
- 더블클릭으로 바로 실행 가능
- Windows 전용

> ⚠️ 처음 실행 시 Windows Defender에서 경고가 뜰 수 있습니다. "추가 정보" → "실행"을 클릭하세요.

---

## ⚠️ 주의사항

**이 프로그램은 교육 목적으로만 사용하세요.**
- 학교 정책을 확인하고 사용하세요
- 출석 인정 기준을 확인하세요
- 자동화 감지 시스템이 있을 수 있습니다

## �️ GUI 버전 (권장)

### 실행 방법

**방법 1: exe 파일 사용 (권장)**
- `dist/LectureAutoPlayer.exe` 파일을 실행

**방법 2: Python으로 직접 실행**
```bash
pip install -r requirements.txt
python lecture_player_gui.py
```

### 사용법

1. **URL 입력**: 강의 목록 페이지 또는 단일 강의 URL 입력
2. **자동 재생 시작** 버튼 클릭
3. **로그인**: 브라우저가 열리면 수동으로 로그인
4. 자동으로 모든 강의가 재생됩니다

---

## 💻 콘솔 버전

### 실행

```bash
python lecture_player.py
```

### 사용 흐름

1. **로그인 페이지 URL 입력**: 학교 로그인 페이지 주소
2. **수동 로그인**: 브라우저가 열리면 직접 로그인
3. **강의 페이지 URL 입력**: 재생할 강의 페이지 주소
4. **재생 모드 선택**:
   - `1`: 단일 강의만 재생
   - `2`: 자동 연속 재생 (다음 강의로 자동 이동)

---

## ✨ 기능

- **자동 비디오 스킵**: 비디오를 거의 끝(종료 0.5초 전)으로 이동
- **연속 재생**: 강의 목록의 모든 강의를 자동 재생
- **단일/목록 자동 감지**: URL에 따라 자동으로 모드 선택

---

## 🔧 동작 원리

1. **로그인**: Selenium으로 브라우저 제어, 수동 로그인 대기
2. **강의 탐색**: 페이지에서 강의 링크 자동 추출
3. **비디오 스킵**: JavaScript로 비디오 끝으로 이동
4. **완료 확인**: 비디오 종료 후 다음 강의로 이동

## 🛠️ 트러블슈팅

### 비디오를 찾을 수 없습니다
- 페이지가 완전히 로드될 때까지 기다리세요
- 다른 강의 페이지 URL을 시도하세요

### 로그인이 안 됩니다
- 브라우저에서 수동으로 로그인하세요
- 로그인 후 자동으로 인식됩니다 (또는 30초 후 진행)

### exe 실행 시 오류
- Chrome 브라우저가 설치되어 있어야 합니다
- 인터넷 연결을 확인하세요
