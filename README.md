# 학교 강의 자동 재생 프로그램

학교 강의 사이트에서 동영상을 자동으로 재생하고 다음 강의로 넘어가는 프로그램입니다.

## ⚠️ 주의사항

**이 프로그램은 교육 목적으로만 사용하세요.**
- 학교 정책을 확인하고 사용하세요
- 출석 인정 기준을 확인하세요
- 자동화 감지 시스템이 있을 수 있습니다

## 🚀 설치

## 💻 사용법

### 실행

```bash
python lecture_player.py
```

### 사용 흐름

1. **로그인 페이지 URL 입력**: 학교 로그인 페이지 주소
2. **수동 로그인**: 브라우저가 열리면 직접 로그인
3. **강의 페이지 URL 입력**: 재생할 강의 페이지 주소
4. **재생 모드 선택**:
   - `1`: 단일 강의만 재생
   - `2`: 자동 연속 재생 (다음 강의로 자동 이동)

## ✨ 기능

### 자동 비디오 스킵
- iframe 내 video 태그 자동 감지
- JavaScript로 비디오를 거의 끝(종료 0.5초 전)으로 이동
- 비디오 종료 후 자동으로 다음 강의 탐색

### 연속 재생
- "다음" 버튼 자동 감지 및 클릭
- 수동 URL 입력으로 다음 강의 지정 가능
- 최대 재생 개수 설정 가능

## 🔧 동작 원리

1. **로그인**: Selenium으로 브라우저 제어, 수동 로그인 대기
2. **iframe 탐색**: 페이지 내 모든 iframe 검색
3. **비디오 찾기**: 각 iframe에서 `<video>` 태그 탐색
4. **JavaScript 실행**:
   ```javascript
   var vid = document.querySelector('video');
   vid.play();
   vid.currentTime = vid.duration - 0.5;
   ```
5. **다음 강의**: "다음" 버튼 클릭 또는 URL 입력

## 🛠️ 트러블슈팅

### iframe을 찾을 수 없습니다
- 페이지가 완전히 로드될 때까지 기다리세요
- 다른 강의 페이지 URL을 시도하세요

### 비디오를 찾을 수 없습니다
- 비디오가 iframe 안에 있는지 확인하세요
- F12 개발자 도구로 페이지 구조 확인

### 다음 버튼을 찾을 수 없습니다
- 수동으로 다음 강의 URL을 입력하세요
- 또는 'q'를 입력하여 종료
